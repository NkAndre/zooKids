# 🦁🐵 ZOO KIDS 🐘🦒

Bem-vindo ao **Zoo Kids**! 🎉  
Um aplicativo divertido e educativo sobre o **Zoológico de São Paulo**, feito especialmente para crianças aprenderem brincando! 🌈✨

---

## 📱 Sobre o App

No **Zoo Kids**, as crianças podem conhecer animais incríveis, explorar curiosidades e se divertir enquanto aprendem sobre a natureza. 🌍💚

---

## 🌟 O que tem no app?

✅ Animais fofos e incríveis  
✅ Curiosidades  
✅ Interface colorida e fácil de usar  
✅ Aprendizado para crianças  
✅ Diversão garantida 🎈
